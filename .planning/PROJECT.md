# Sistema de Préstamos y Cobranza

## Resumen del Proyecto
Sistema integral para la gestión de préstamos, cobranzas, ahorros y gastos. Permite el seguimiento de clientes, cronogramas de pago y estados financieros.

## Stack Tecnológico
- **Backend:** Node.js, Express
- **Frontend:** EJS, Bootstrap, Vanilla JS
- **Base de Datos:** MySQL
- **Utilidades:** PDFKit, ExcelJS

## Objetivos
- Facilitar la administración de carteras de préstamos.
- Automatizar el cálculo de intereses y cuotas.
- Proporcionar reportes financieros detallados.
